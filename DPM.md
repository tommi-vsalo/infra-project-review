# DPM - Dumb Packet Manager

## Project Overview
The DPM project seeks to create an tool which sets up command-line tools automatically for use in workshops, CTF competitions etc. The end result was a downloadable tool which could download other tools or apply profiles reliably.

The project was tested on a Debian 13 virtual machine.
